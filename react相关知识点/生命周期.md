使用注意事项：

**constructor函数里禁止调用setState函数。**
```jsx
//千万不要这么做，更新prop中的color，并不会影响state,这就会产生bug
constructor(props) {
 super(props);
 // 不要这样做
 this.state = { color: props.color };
}
```
