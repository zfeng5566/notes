# 类数组

**类数组拥有一个`length`属性和若干索引属性的任意对象。**
```js
{
    0:"a",
    1:"b",
    2:"c",
    length:3
}
```
**一个类数组转成数组的方式**
```js
//第一种方式
function run(){
    return Array.prototype.slice.call(arguments)
}
//第二种方式
Array.prototype.concat.apply([],arguments);
//第三种方式
Array.from(arguments);
//第四种方式
const arr1 = [...arguments];
```
